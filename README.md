# CitaMedicaBeta

Sistema de gestión de citas médicas con chatbot integrado (WhatsApp).

## Stack

- **Frontend**: React + TypeScript + Tailwind CSS v4
- **Backend**: Node.js + Express + MongoDB
- **Chatbot**: BuilderBot + Baileys (WhatsApp)
- **Infra**: VPS + PM2 + Nginx + Tailscale

## Estructura del Proyecto

```
CitaMedicaBeta/
├── frontend/          # React app
├── backend/           # Express API
├── docs/              # Documentación (no trackeada por git)
│   ├── ops/           # Deploy, seguridad
│   ├── chatbot/       # Config del chatbot
│   ├── dev/           # Guías de desarrollo

```
